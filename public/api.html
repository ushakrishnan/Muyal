This page shows the available API endpoints for integrating with the Muyal AI Agent.

POST /api/chat

Description: Send a message to the AI agent and get a response

Request Body:

{
  "message": "Hello, how are you?",
  "conversationId": "user-123" // optional, defaults to 'web-user'
}

Response:

{
  "response": "I'm doing well, thank you! How can I help you today?",
  "conversationId": "user-123"
}

GET /api/conversation/:id

Description: Get conversation history for a specific conversation ID

Response:

{
  "history": [
    {"role": "user", "content": "Hello"},
    {"role": "assistant", "content": "Hi there!"}
  ]
}

DELETE /api/conversation/:id

Description: Clear conversation history for a specific conversation ID

Response:

{"success": true}

GET /api/health

Description: Health check endpoint

Response:

{
  "status": "ok",
  "timestamp": "2025-01-01T12:00:00.000Z"
}

ðŸ“– Integration Examples

JavaScript/Fetch

async function sendMessage(message, conversationId = 'web-user') {
  const response = await fetch('/api/chat', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ message, conversationId })
  });
  
  const data = await response.json();
  return data.response;
}

cURL

curl -X POST http://localhost:3978/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!", "conversationId": "user-123"}'

Python

import requests

response = requests.post('http://localhost:3978/api/chat', 
  json={
    'message': 'Hello!',
    'conversationId': 'user-123'
  }
)

data = response.json()
print(data['response'])
